
# TalkHub - Landing Page

Talkhub, we believe that learning should be accessible to all, anytime, anywhere. .
 
## How to use


### 1\. Install dependencies

```
yarn install
# or
npm install
```

### 2\. Run Dev Server

```
yarn dev
# or
npm run dev
```
